Développement d'une macro VBA pour Amaline Assurances

Amaline Assurances est une entreprise de courtage du groupe Groupama. L'entreprise a 3 types de partenariats, le partenaire principal Renault/Dacia où ils réalisent des contrats auto/habitation pour Groupama. Le contrat emprunteur pour GGVIE (Groupama GAN Vie) et le contrat juridique/animaux pour GPJ (Groupama Protection Juridique).

Le but de ce stage était de créer une macro VBA pour le fichier des accords clients des conseillers pour les responsables d'équipes de chez Amaline Assurances. Le but de la macro était de copier les données des différents fichiers conseiller pour les coller dans un seul fichier, ensuite d'harmoniser les données puis de mettre à jour certains indicateurs comme qui a exécuté en dernier la macro et à quel date, le nombres de lignes dans le fichier, etc.
